Snowball Stemmers

Snowball is actually a language for creating stemmers, and was 
added to NLTK version 2.0b9 as the SnowballStemmer class. 
The NLTK Snowball stemmer currently supports the following languages:

    Danish
    Dutch
    English
    Finnish
    French
    German
    Hungarian
    Italian
    Norwegian
    Porter
    Portuguese
    Romanian
    Russian
    Spanish
    Swedish
